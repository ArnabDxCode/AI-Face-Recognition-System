# 🤖 AI Face Recognition System

A real-time cyber-style Face Recognition system built using Python and OpenCV.

- Smooth fade-in scanning animation
- Dark cyber HUD overlay
- Stabilized face detection box
- Face recognition with confidence %
- Voice greeting system
- Automatic attendance logging
- FPS counter
- Futuristic UI design

## 🛠 Technologies Used

- Python
- OpenCV
- LBPH Face Recognizer
